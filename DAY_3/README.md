# 🚀 Day 3 - Simple Calculator

Welcome to **Day 3** of my **100 Days, 100 Python Projects** challenge!

This project is a simple command-line calculator built using Python. It allows users to perform various arithmetic operations on two numbers and also displays the results of all supported operations.

---

## 📌 Project Overview

The program asks the user to enter:

- 🔢 First Number
- 🔢 Second Number
- ➕ Arithmetic Operation

The calculator supports multiple mathematical operations and safely handles division by zero.

---

## ✨ Features

- Accepts decimal and whole numbers
- Supports multiple arithmetic operations
- Displays the result of the selected operation
- Shows results for all supported operations
- Handles division by zero gracefully
- Beginner-friendly Python project

---

## 🛠️ Technologies Used

- Python 3

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Clone this repository or download the project.
3. Open the terminal in the project folder.
4. Run the program:

```bash
python main3.py
```
