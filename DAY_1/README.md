# 🚀 Day 1 - Welcome Message Generator

Welcome to **Day 1** of my **100 Days, 100 Python Projects** challenge!

This is a simple Python program that takes user input and generates a personalized welcome message along with the current system time.

---

## 📌 Project Overview

The program asks the user for:

- 👤 Name
- 🎯 Favorite Hobby
- 🎨 Favorite Color

It then displays a friendly welcome message using the entered information and shows the current time.

---

## ✨ Features

- Accepts user input
- Converts input to uppercase
- Displays the current system time
- Uses formatted strings (f-strings)
- Beginner-friendly Python project
- Simple and easy to understand

---

## 🛠️ Technologies Used

- Python 3
- `time` module

## ▶️ How to Run

1. Make sure Python 3 is installed.
2. Clone this repository or download the project.
3. Open the terminal in the project folder.
4. Run the program:

```bash
python day1.py
```
