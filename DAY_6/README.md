# 🚀 Day 6 - Basic Math Quiz Game

Welcome to **Day 6** of my **100 Days, 100 Python Projects** challenge!

This project is a command-line Math Quiz Game built with Python. It generates random arithmetic questions, evaluates the user's answers, keeps track of the score, and provides feedback based on the final performance.

---

## 📌 Project Overview

The program allows the user to choose how many questions they want to answer. For each round, it randomly generates a math question using basic arithmetic operators and checks whether the user's answer is correct.

At the end of the quiz, the program displays the final score along with a performance message.

---

## ✨ Features

- 🎲 Generates random math questions
- ➕ Supports multiple arithmetic operations:
  - Addition (`+`)
  - Subtraction (`-`)
  - Multiplication (`*`)
  - Division (`/`)
- 📝 User chooses the number of quiz questions
- ✅ Checks answers instantly
- 📊 Tracks and displays the final score
- 🎉 Gives personalized feedback based on performance
- 🖥️ Beginner-friendly command-line application
