# 🚀 Day 4 - Number Comparison Tool

Welcome to **Day 4** of my **100 Days, 100 Python Projects** challenge!

This project is a menu-driven Python program that compares two or three numbers, identifies the largest and smallest values, and determines whether each number is positive, negative, or zero.

---

## 📌 Project Overview

The program provides a simple menu with the following options:

1. Compare 2 Numbers
2. Compare 3 Numbers
3. Exit

Based on the user's choice, the program performs number comparisons and displays meaningful results. It also allows users to perform multiple comparisons without restarting the program.

---

## ✨ Features

- 📊 Compare two numbers
- 📈 Compare three numbers
- 🏆 Find the largest and smallest number
- 🤝 Detect equal numbers
- ➕ Check whether each number is:
  - Positive
  - Negative
  - Zero
- 🔄 Menu-driven interface
- ♻️ Allows multiple comparisons in a single run
- ❌ Handles invalid menu choices gracefully
- 🚪 Exit option to close the program
