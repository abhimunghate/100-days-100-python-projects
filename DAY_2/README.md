# 🚀 Day 2 - Personalized Greeting Program

Welcome to **Day 2** of my **100 Days, 100 Python Projects** challenge!

This Python program collects basic information from the user and generates a personalized greeting while also calculating the user's approximate age in years, months, weeks, and days.

---

## 📌 Project Overview

The program asks the user for:

- 👤 Name
- 🎂 Birth Year
- 🎈 Current Age
- 🎨 Favorite Color

Using the birth year, it calculates the user's approximate age in:

- Years
- Months
- Weeks
- Days

Finally, it displays a personalized greeting with all the collected information.

---

## ✨ Features

- Accepts user input
- Converts the user's name to uppercase
- Calculates approximate age from the birth year
- Displays age in months, weeks, and days
- Uses formatted strings (f-strings)
- Beginner-friendly Python project
